# 🐱🐶 Cat vs Dog Image Classification using CNN

A Deep Learning project that uses **Convolutional Neural Networks (CNNs)** to classify images of cats and dogs.
This project demonstrates image preprocessing, model building, training, evaluation, and prediction using **TensorFlow and Keras**.



## 📌 Project Overview

Convolutional Neural Networks (CNNs) are powerful deep learning models designed for image data.
In this project, we build a CNN-based classifier that can distinguish between:

* 🐱 Cats
* 🐶 Dogs

The model is trained on the popular Kaggle dataset containing **25,000 images**.



## 📂 Dataset

* Dataset: Kaggle Dogs vs Cats Dataset
* Structure:

  ```
  dog-vs-cat-classification/
      ├── cat/
      └── dog/
  ```
* Each folder contains **12,500 images**.
